Development of a Manufacturer AI-Supported Web Platform for IoT Penetration Testing Project

TÜBİTAK–2209-A University Students Research Projects Support Program within the scope of this project, we aim to develop a unique AI-supported web platform to ensure the security of IoT devices. IoT (Internet of Things) devices stand out in daily life with their ability to collect and process data synchronously. However, the security of these devices is a major concern due to many vulnerabilities and security gaps.

Our Project Team

Our project is conducted by the academic advisor from Süleyman Demirel University, Assoc. Prof. Dr. Utku Köse and project executives Zehra Hatipoğlu, Sedanur Ceylan and Büşra Yaman. As a team, we develop innovative and effective solutions to enhance the security of IoT devices.

Project Aim and Objectives

The main aim of our project is to develop a web platform using artificial intelligence techniques to create unique malware datasets that can be used for educational purposes and penetration testing of IoT devices. In this context, previously collected datasets are processed using the Generative Adversarial Networks (GAN) deep learning technique to create new datasets that are very close to reality. These unique datasets will be available for users to use for educational and penetration testing purposes.

Original Value

Our project differentiates itself from other studies with GAN-based approaches developed to detect and solve the security vulnerabilities of IoT devices. The datasets created with GAN models contain real-like but entirely unique malware examples. This way, users will have the opportunity to simulate real attack scenarios and practice.

Project Impact

The developed web platform can be used by both individual users and institutions for education and awareness purposes. Through this platform, users will be able to test the security of IoT devices, identify security vulnerabilities, and be prepared against possible attacks.

With this project, we aim to make significant contributions to the field of IoT security and create a more secure IoT ecosystem in the future.

To learn more about our project and follow our work, you can visit our GitHub repository.
